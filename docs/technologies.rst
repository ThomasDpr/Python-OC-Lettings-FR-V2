Technologies et langages
========================

Backend
-------

* **Python 3.11**
* **Django 3.0** : Framework web
* **Gunicorn** : Serveur WSGI pour production

Base de données
---------------

* **SQLite3** : Base de données de développement et production

Tests et qualité
----------------

* **pytest** : Framework de tests
* **pytest-django** : Plugin pytest pour Django
* **pytest-cov** : Couverture de tests
* **flake8** : Linting Python (PEP8)

Monitoring
----------

* **Sentry** : Surveillance des erreurs et logging

Déploiement
-----------

* **Docker** : Containerisation
* **Docker Hub** : Registre d'images
* **GitHub Actions** : CI/CD
* **Render** : Hébergement production
* **WhiteNoise** : Gestion des fichiers statiques

Documentation
-------------

* **Sphinx** : Génération de documentation
* **Read the Docs** : Hébergement de documentation
